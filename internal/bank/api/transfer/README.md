# Transfer API Quest Zone

Welcome to the workshop! This directory is your workspace for the **[Go Bank Transfer Quest](../../../challenges/bank/README.md)**. 

Your mission is to bring the `POST /v1/transfers` endpoint to life!

## Your Roadmap to Success

Don't start coding from scratch—we learn by reading great code! 

1. **Study the Masterpiece:** Read the fully functional implementation over in the [api/account/](../account) directory. You'll be using those exact same patterns here.
2. **Define the Shape:** Open [types.go](types.go) and define the JSON request and response structures.
3. **Build the Engine:** Open [handler.go](handler.go) and follow the 5 guided `TODO`s to implement the HTTP logic.
4. **Prove it Works:** Open [handler_test.go](handler_test.go) and complete the table-driven test cases to verify you handled all the edge cases perfectly.

*(Tip: Keep your changes focused! You only need to touch the areas marked with TODOs to complete this quest.)*
