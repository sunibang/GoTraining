# Challenge 03: Error Handling

## Goal
Implement a function that returns an error and handle it in your test. This challenge tests your understanding of Go error handling patterns.

## Tasks
1. Define a function `Divide(a, b float64) (float64, error)` that divides `a` by `b`.
2. Return a custom error if the divisor is `0`.
3. Handle the error in your test.

## Run Tests
```bash
go test ./internal/challenges/basics/03-error-handling/...
```
